# JH Design System Development Sandbox

An example project you can copy to start using [The Jack Henry Design System](https://main--68f8e6a25b256d0ef89b13e6.chromatic.com/?path=/docs/what-s-new-v2-release--docs) today!

### Prerequisites

- Node.js 18+
- Yarn

### Setup

Copy this directory into you project.

```bash
# Install dependencies
yarn install

```

Run the app with a Banno FI's theme by providing the institution id:
```bash
INSTITUTION_ID=5db40b7a-ee76-42b3-acb9-eb7045011c2a yarn dev
```

Or with the default Jack Henry theme:
```bash
yarn dev
```

### Building components

The example component provided demonstrates some of the available Design System components, and icons and also a simple example of how to create a Single Page Application using [Lit](https://lit.dev).
